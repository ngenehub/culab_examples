# CuPy Benchmarks

This directory contains **Python (CuPy) benchmark scripts** used to measure **GPU execution time** for CuLab performance comparisons.

The scripts generate **execution time CSV files** that are later used by LabVIEW benchmark VIs.

---

## Requirements

⚠️ **Conda is required**

Conda must be installed before running any benchmarks.

---

## Running Benchmarks

After installing Conda and following the instructions described in **CuPy_Benchmark_Setup.pdf**, run:

```bash
Python_Run_ALL.bat
